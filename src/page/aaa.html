1. HTML5 - โครงสร้างและเซแมนติก

HTML ไม่ใช่ภาษาโปรแกรม แต่เป็นภาษา Markup ที่บอก Browser ว่าข้อมูลไหนคืออะไร

พื้นฐานที่ต้องแม่น (Basics)

<!-- โครงสร้างพื้นฐาน -->
<!DOCTYPE html>
<html lang="th">
  <head>
    <title>ชื่อเว็บ</title>
  </head>
  <body>
    <h1>หัวข้อหลัก</h1>
    <p>ย่อหน้า</p>
    <a href="url">ลิงก์</a>
    <img src="image.jpg" alt="คำอธิบายภาพ">
  </body>
</html>

ระดับกลาง: Semantic HTML (สำคัญต่อ SEO มาก)

แทนที่จะใช้ <div> ทั้งเว็บ ควรใช้ Tag ที่มีความหมาย:

  • <header>: ส่วนหัวของหน้าหรือบทความ
  • <nav>: ส่วนเมนูนำทาง
  • <main>: เนื้อหาหลัก (มีได้เพียง 1 เดียวในหน้านั้น)
  • <article>: เนื้อหาที่แยกชิ้นได้เอง (เช่น โพสต์บล็อก)
  • <section>: กลุ่มเนื้อหาที่เกี่ยวข้องกัน
  • <footer>: ส่วนท้ายของหน้า
Pro Tip: การใช้ Semantic HTML ช่วยให้ Search Engine (Google) เข้าใจโครงสร้างเว็บคุณได้ดีขึ้น และช่วยให้เครื่องอ่านหน้าจอ (Screen Reader) ทำงานได้ถูกต้อง

ระดับสูง: Forms & Meta

2. CSS3 - การออกแบบและเลย์เอาต์

The Box Model (หัวใจของ CSS)

ทุก Element คือกล่องที่ประกอบด้วย:

ชั้น คำอธิบาย
Content เนื้อหาข้างใน (ข้อความ/รูป)
Padding ระยะห่างจากเนื้อหาถึงขอบกล่อง (ด้านใน)
Border เส้นขอบ
Margin ระยะห่างระหว่างกล่องนี้กับกล่องอื่น (ด้านนอก)

Modern Layout: Flexbox vs Grid

Flexbox (1 มิติ) 
.container {
  display: flex;
  justify-content: center; /* แนวนอน */
  align-items: center;    /* แนวตั้ง */
  flex-direction: row;
}
Grid (2 มิติ) 
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

Advanced CSS: Variables & Animations

/* การใช้ตัวแปร */
:root {
  --main-blue: #007bff;
}

.button {
  background-color: var(--main-blue);
  transition: all 0.3s ease;
}

.button:hover {
  transform: scale(1.1);
  filter: brightness(1.2);
}

3. Modern JavaScript (ES6+)

Variables & Logic

// เลิกใช้ var ให้ใช้ let และ const
const pi = 3.14; 
let score = 0;

// Arrow Function (เขียนสั้นลง)
const welcome = (name) => `สวัสดีคุณ ${name}`;

// Template Literals
console.log(`ค่าพายคือ ${pi}`);

การจัดการ DOM (ทำให้เว็บมีชีวิต)

// การเลือกและเปลี่ยนเนื้อหา
const btn = document.querySelector('#myBtn');
const title = document.querySelector('h1');

btn.addEventListener('click', () => {
  title.innerText = 'ถูกคลิกแล้ว!';
  title.style.color = 'red';
});

ระดับสูง: Asynchronous & API

ใช้ async/await เพื่อดึงข้อมูลจาก Server โดยไม่ต้องรอหน้าเว็บโหลดใหม่

async function getUserData() {
  try {
    const response = await fetch('https://api.example.com/user');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error("เกิดข้อผิดพลาด:", error);
  }
}

// เรียกใช้งาน
getUserData();
